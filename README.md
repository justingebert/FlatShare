## start developing

- pull Repo -> npm i
- Open Terminal and type: tsc -> press ENTER (if command not found -> npm install typescript -g // on mac maybe with sudo for premissions)
- Open another Terminal and type: npm run start (This starts the Server)
- Ready to go
